# Google Blogger API

The Google Blogger API v3 allows you to integrate Blogger content with your application. You can create, read, update, and delete blogs, posts, pages, comments, and user information using RESTful operations with OAuth 2.0 authentication.

## Artifacts

- **OpenAPI**: [openapi/blogger.yml](openapi/blogger.yml) - OpenAPI 3.1.0 specification for the Google Blogger API v3.
- **JSON Schema**: [json-schema/blogger.json](json-schema/blogger.json) - JSON Schema (draft 2020-12) for Blogger post resources.
- **JSON-LD**: [json-ld/blogger.jsonld](json-ld/blogger.jsonld) - JSON-LD context mapping Blogger resources to Schema.org vocabulary.
